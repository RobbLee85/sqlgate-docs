## 공통
| 이름 | 바로 가기 키 | 이름 | 바로 가기 키 |
|  ---  |  ---  |  ---  |  ---  |
| 새로 만들기 | Ctrl + N | 잘라내기 | Ctrl + X |
| 열기 | Ctrl + O | 복사 | Ctrl + C |
| 저장 | Ctrl + S | 붙여넣기 | Ctrl + V |
| 다른 이름으로 저장 | Ctrl + Shift + S | 모두 선택 | Ctrl + A |
| 실행 취소 | Ctrl + Z | 삭제 | Del |
| 다시 실행 | Shift + Ctrl + Z |  |  |

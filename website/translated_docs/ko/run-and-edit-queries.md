## 쿼리를 실행하고 편집하기

**쿼리를 생성하고 실행하는 방법은 아래와 같습니다.**
1. 오라클 데이터베이스에 접속합니다.
2. 주 메뉴 **파일**>**새로 만들기**>**SQL 편집기**를 선택합니다. 또는 새로 만들기 도구 모음에서 [SQL 편집기]를 클릭하거나 Ctrl+N를 누릅니다.
3. SQL을 작성합니다.
4. [실행] 버튼을 클릭하거나 F5를 누릅니다.
5. 쿼리 실행 결과를 확인합니다.

테이블 스크립트 내보내기

테이블 스크립트 내보내기를 설명합니다.

  1. 데이터베이스에 접속합니다.
  2. 주 메뉴 도구>보고서>스크립트 내보내기를 선택합니다. 또는 도구 도구 모음에서 스크립트 내보내기를 클릭합니다.
  3. 스크립트 내보내기 창이 열립니다.
  4. 개체 목록에서 개체를 선택합니다.
  5. 스키마와 개체를 선택합니다.
  6. 스크립트 보기 탭에서 스크립트 내보내기를 클릭합니다.
  7. 내보낼 폴더를 선택합니다.
  8. 파일로 내보내기 버튼을 클릭합니다.
  9. 로그 창에서 실행 결과를 확인합니다.

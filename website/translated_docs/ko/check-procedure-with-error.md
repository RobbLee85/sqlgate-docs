## 오류가 난 프로시저 확인하기

오류가 난 프로시저 확인하기를 설명합니다.

1. 데이터베이스에 접속합니다.
2. 주 메뉴 보기>개체 탐색기를 실행합니다.
3. 프로시저 개체를 클릭합니다.
4. 오류가 난 프로시저를 확인합니다.

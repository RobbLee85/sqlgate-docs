ButtonInstructionShortcut KeysAction

Run SubQuery blockAlt+F5Run the SubQuery block.
StopESCStop running SQL.
Run and Modify Current SQLF6Run current SQL and modify the data on the grid.
Run Current SQL Execution PlanF7Run current SQL Execution Plan.
Block SQL Execution PlanAlt+F7Run subquery block SQL Execution Plan.
SQL HistoryF8Run SQL History.
Show GridF11Toggle/Collapse grid.
SQL AlignmentF9Align SQL in SQL Editor.
SQL Alignment OptionsSet SQL Alignment options.
Move Tabs to RightMove the tab located at the bottom to the right.

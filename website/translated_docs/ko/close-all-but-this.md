## 현재 창 이외에 모두 닫기

현재의 작업 창 이외에 모든 탭 닫기를 설명합니다.

1. 데이터베이스에 접속합니다.
2. 현재 탭에서 마우스 오른쪽을 클릭하고 이 창을 제외하고 모두 닫기를 선택합니다.
![현재 창 제외하고 모두 닫기](https://s3.ap-northeast-2.amazonaws.com/sqlgate-manual-content/BA6B7E9F98FF4D0B477401E5C8D4D4E6.jpg)
3. 실행 결과를 확인 합니다.

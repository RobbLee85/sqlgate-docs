## 데이터 추가하기

결과 창에서 데이터를 추가하는 방법을 설명합니다.

1. 결과 창 네비게이터의 레코드 추가 버튼을 클릭합니다.
2. 맨 아래 빈 레코드가 추가됩니다.
3. 데이터를 입력합니다.
4. 레코드 저장 버튼을 클릭합니다.
5. 새로 고침 버튼을 클릭하여 실행 결과를 확인합니다.
